# Hello - Nest

## Description

Created a movie API using NestJS, TypeScript and GraphQL.
Conducted simple unit and e2e testing using Nest Test.

## Installation

```bash
$ npm install
```

# Dependencies
- NodeJS
- JavaScript
- TypeScript
- GraphQL
- NestJS
- RESTful API communication with NoSQL API
- Nest Test

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

Current Test Coverage is at 100%
